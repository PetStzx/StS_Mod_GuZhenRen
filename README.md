# Slay the Spire Mod - Reverend Insanity(蛊真人)

## 简体中文

### 简介

这是一个为 [**杀戮尖塔**](https://store.steampowered.com/app/646570/Slay_the_Spire/) 制作的模组。
本项目将《蛊真人》的主角 —— **古月方源** 作为可玩角色加入到游戏中。

### 创意工坊地址

你可以直接在 Steam 创意工坊订阅并安装：[点击前往 Steam 创意工坊](https://steamcommunity.com/sharedfiles/filedetails/?id=3701087103)

### 前置需求

在运行此模组之前，请确保你已经安装了以下基础依赖模组：

* [BaseMod](https://github.com/daviscook477/BaseMod/releases)
* [ModTheSpire](https://github.com/kiooeht/ModTheSpire)
* [StSLib](https://github.com/kiooeht/StSLib)

## English

### Introduction

This is a mod for [**Slay the Spire**](https://store.steampowered.com/app/646570/Slay_the_Spire/), which adds **Gu Yue Fang Yuan** (from the novel *Reverend Insanity*) as a playable character.

### Steam Workshop

You can subscribe to and install this mod via the Steam Workshop:[Visit Steam Workshop Page](https://steamcommunity.com/sharedfiles/filedetails/?id=3701087103)

### Requirements

Please ensure you have the following dependency mods installed:

* [BaseMod](https://github.com/daviscook477/BaseMod/releases)
* [ModTheSpire](https://github.com/kiooeht/ModTheSpire)
* [StSLib](https://github.com/kiooeht/StSLib)
