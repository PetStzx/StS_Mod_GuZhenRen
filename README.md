# Slay the Spire Mod - GuZhenRen

## Introduction ##
This is a mod for [**Slay the Spire**](https://store.steampowered.com/app/646570/Slay_the_Spire/), which adds Gu Yue Fang Yuan as a playable character.<br>

### Requires:
 * [BaseMod](https://github.com/daviscook477/BaseMod/releases)
 * [ModTheSpire](https://github.com/kiooeht/ModTheSpire)
 * [StSLib](https://github.com/kiooeht/StSLib)
